## Rhetorical

| Class | Description |
| ------ | ------ |
| [Rhetorical](Question/Rhetorical.md) | A rhetorical question is a special type of question in which the answer is part of the question itself. This is useful for cases where the answer to a question may optionally be given by the entity asking the question. This means that a question can be conditionally answered while still retaining its monadic structure as the question isn't unwrapped to its answer. |
